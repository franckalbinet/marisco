OSPAR

Data pipeline (handler) to convert OSPAR data (source) to NetCDF format.

The OSPAR Environment database is provided as a Microsoft Access database. Mdbtools (https://github.com/mdbtools/mdbtools) can be used to convert the tables of the Microsoft Access database to .csv files on Unix-like OS.

Example steps: 1. Download data. 2. Install mdbtools via VScode Terminal

```
sudo apt-get -y install mdbtools
````

  1. Install unzip via VScode Terminal

    sudo apt-get -y install unzip

  2. In VS code terminal, navigate to the marisco data folder

    cd /home/marisco/downloads/marisco/_data/accdb/mors_19840101_20211231

  3. Unzip MORS_ENVIRONMENT.zip

    unzip MORS_ENVIRONMENT.zip

  4. Run preprocess.sh to generate the required data files

    ./preprocess.sh MORS_ENVIRONMENT.zip

  5. Content of ‘preprocess.sh’ script. ``` #!/bin/bash

    # Example of use: ./preprocess.sh MORS_ENVIRONMENT.zip unzip \(1 dbname=\)(ls .accdb .mdb) mkdir csv for table in \((mdb-tables -1 "\)dbname”); do echo “Export table \(table" mdb-export "\)dbname” “\(table" > "csv/\)table.csv” done ```
